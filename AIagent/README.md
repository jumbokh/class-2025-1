## 《智能系统开发实践》課程安排（12 次 × 4 課時）

### 第1次：導論與課程定位

* 主題：智能系统开发实践課程簡介
* 次題：

  * AI Agent 概念與核心特性 (第1章 1.1–1.5)
  * 智能系统实践課程目標與評估方式
  * 案例展示：日程管理 AI Agent

---

### 第2次：AI Agent 基礎原理

* 主題：AI Agent 的理論基礎與技術支撐
* 次題：

  * AI Agent 的歷史演進與應用場景 (1.2, 1.3, 1.4)
  * 理論支撐：人工智慧、代理理論、控制與博弈 (1.6)
  * 技術支撐：數據、機器學習、NLP、視覺、雲計算 (1.7)

---

### 第3次：AI Agent 與 LLM

* 主題：大語言模型在智能體中的角色
* 次題：

  * 主流 LLM 介紹 (2.1)
  * LLM 的原理與作用 (2.2)
  * LLM 在 AI Agent 中的功能 (2.3–2.4)
  * LLM 與用戶介面的結合 (2.5)

---

### 第4次：工具平台實踐（扣子平台）

* 主題：扣子平台的使用與工作流設計
* 次題：

  * 扣子平台概述與功能 (3.1.1–3.1.2)
  * 案例演示：調用大語言模型進行對話
  * 學生實操：設計一個小型對話式 Agent

---

### 第5次：創作類 AI Agent（實驗1）

* 主題：AI Agent 創作實踐 (第4章)
* 實驗項目：歌曲創作 AI Agent (4.2)
* 次題：

  * 需求分析與功能目標
  * 扣子平台設計步驟
  * 學生實作：生成一首歌曲歌詞

---

### 第6次：故事與教育應用（實驗2）

### 第7次：出題系統（實驗3）
